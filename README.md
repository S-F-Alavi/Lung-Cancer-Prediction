# Lung-Cencer-Prediction
Lung cancer prediction project using RapidMiner
# پیش‌بینی سرطان ریه

این پروژه با استفاده از نرم‌افزار RapidMiner، احتمال ابتلا به سرطان ریه را پیش‌بینی می‌کند.

## فایل‌ها
- چهار فایل فرآیند RapidMiner (.rmp) در پوشه rmp-files/
- دیتاست مورد استفاده برای آموزش و تست مدل در پوشه dataset/

## نحوه اجرا
برای اجرای پروژه، فایل‌های .rmp را با RapidMiner Studio باز کنید.

## توضیحات اضافی
- هر فایل .rmp مربوط به یک الگوریتم یا مرحله خاص پروژه است.
- فایل دیتاست شامل داده‌های آموزشی و آزمایشی است.
